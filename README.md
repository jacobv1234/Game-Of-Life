# Game-Of-Life
Conway's Game of Life simulator with a powerful rule modification tool

## Install
1. Ensure you have Python 3.12 or later, and Anaconda installed.
2. Set up an Anaconda environment for the software.
3. Download the code (code -> download ZIP) and copy into the environment's directory.
4. Use Anaconda to install the following packages:
- numpy
- matplotlib
- numba (Note: numba cannot be installed via pip, hence the Anaconda environment)

## Launch
Once the install process is complete, run main.py to start the software.
